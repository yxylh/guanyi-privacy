隐私政策

生效日期：2026年06月15日

本隐私政策适用于「观易」移动端应用及相关服务，运营开发者为李辉（以下统称“服务提供者”）。「观易」是一款提供易经经典阅读、罗盘文化工具及占筮符号参考的移动应用，无注册、无登录功能，无需用户提交任何个人账号信息即可使用。

本隐私政策用于如实说明本应用的数据收集、使用、第三方服务接入及用户权益相关规则。

一、应用功能与数据说明

本应用主要功能包括：

  • 易经阅读：浏览六十四卦原文、卦辞及释义，支持卦象收藏（数据保存在用户设备本地）；
  • 罗盘工具：基于设备方向传感器展示传统罗盘界面，用于文化学习与方位参考；
  • 占筮参考：基于易经符号系统起卦并展示卦象与经典释义，占筮记录保存在用户设备本地；
  • 会员服务：通过 Apple App Store 订阅，享受免广告、更多收藏与记录容量等权益。

上述功能产生的收藏、占筮记录等数据均仅存储在用户设备本地，服务提供者不上传、不云端同步上述内容。

二、信息收集与使用

在您下载并使用本应用的过程中，应用及接入的第三方 SDK 可能收集相关设备及使用数据，具体包括：

  • 设备网络协议地址（IP 地址）
  • 应用访问页面、访问时间、使用时长
  • 设备操作系统类型及系统版本
  • 设备型号、网络状态等基础设备信息

本应用不收集、不存储、不上传用户姓名、手机号、身份证、精确地理位置、相册、麦克风、通讯录等任何个人敏感信息，无用户账号体系，不存在个人隐私数据云端留存。

罗盘功能使用设备方向传感器（陀螺仪/磁力计）在本地计算方位，不上传方位或定位数据。

三、第三方 SDK 数据收集说明（友盟官方合规）

本应用集成友盟统计 SDK、友盟广告 SDK，用于应用数据统计、崩溃分析、合规广告展示，相关数据采集严格遵循《友盟+隐私政策》。

1. 采集内容：仅在用户授权 iOS 跟踪权限（ATT）后，SDK 匿名获取IDFA 广告标识符、IDFV、设备基础标识、网络信息、应用运行日志、匿名崩溃数据；

2. 采集用途：用于应用数据分析、服务优化、BUG 修复、普通用户合规广告投放与广告效果统计。已开通会员的用户将免除广告展示，SDK 不会对会员用户进行广告投放及广告相关数据统计；

3. 授权规则：用户拒绝广告跟踪授权时，本应用及友盟 SDK 将停止获取 IDFA 及广告相关数据，仅保留基础统计能力，不采集任何广告追踪信息；

4. 友盟严格遵循最小必要原则，仅采集服务必需的匿名设备数据，不会采集用户隐私信息；

5. 友盟官方隐私政策链接：https://www.umeng.com/page/policy

四、Cookie 与跟踪技术

本应用及接入的第三方 SDK 可使用 Cookie、像素标签及同类跟踪技术，用于保障应用基础运行、数据统计及广告服务稳定。所有非必要跟踪行为，均在获取用户授权后执行。

六、第三方数据访问与共享

本应用仅向友盟 SDK 传输匿名化、去标识化的设备统计数据，用于应用优化及广告服务，不会向任何第三方出售、泄露、共享可定位用户身份的信息。所有对外数据传输均遵循合规加密及匿名化处理规则。

七、数据留存规则

本应用无用户个人数据云端留存。第三方 SDK 匿名统计数据按以下规则留存：

  • 设备自动采集数据：最长留存 24 个月
  • 匿名聚合统计数据：永久留存（无法识别任何个人身份）
  • 合规所需数据：按法律法规要求留存

用户设备本地的收藏、占筮记录等数据，在用户卸载应用或主动删除后即清除。

八、退出与停止数据收集权利

您可通过卸载本应用、在手机系统设置关闭应用跟踪权限的方式，永久停止应用后续数据收集。卸载应用后，应用将不再采集任何设备数据。

九、未成年人保护

本应用面向全年龄段用户提供文化学习与工具服务，不针对未成年人推送定向内容，不会主动收集未成年人相关数据。未成年人请在监护人陪同下使用本应用。

十、信息安全保障

服务提供者采用行业标准的加密、防火墙、访问管控技术，保障设备匿名数据的传输与存储安全，防止数据泄露、篡改、丢失。
